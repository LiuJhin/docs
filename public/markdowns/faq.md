# 常见问题 (FAQ)

## 为什么选择 MarkView？

MarkView 专注于提供极致简洁的阅读体验，同时保持高性能和良好的可扩展性。

## 支持哪些 Markdown 语法？

我们使用 `@uiw/react-markdown-preview`，支持标准的 Markdown 语法以及 GFM (GitHub Flavored Markdown)。

- 表格
- 代码高亮
- 任务列表
- 删除线
- 等等

## 如何自定义主题？

项目使用 Tailwind CSS，你可以通过修改 `tailwind.config.ts` (或 `globals.css`) 来自定义颜色和样式。

目前内置了明亮和暗黑两种模式。
